# JobQuestAI

JobQuestAI is an AI-powered Career & Exam Preparation Platform designed to help students, job seekers, and competitive exam candidates prepare efficiently through intelligent tools, live job opportunities, and AI-generated assessments.

## Features

### AI Practice Tests

* AI-generated MCQs using Groq API
* Custom subjects and difficulty levels
* Instant scoring and feedback
* Ranking and performance tracking

### Live Jobs Portal

* Live job listings using JSearch API
* Government and private sector jobs
* Job details modal
* Direct apply links
* Category-based filtering

### Competitive Exams Hub

* CSS
* PMS
* PPSC
* FPSC
* NTS
* MDCAT

Includes syllabus information, exam patterns, and official resources.

### User Dashboard

* Test history
* Performance analytics
* Ranking system
* Subject-wise progress tracking

### Admin Panel

* Manage jobs
* Manage tests
* User monitoring
* Content management

### Firebase Integration

* Authentication
* Firestore Database
* Secure data storage

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* Firebase Authentication
* Firebase Firestore
* Groq API
* JSearch API

## Project Goal

To provide a single platform where students can prepare for competitive exams, practice AI-generated tests, explore educational resources, and discover real-time job opportunities.
